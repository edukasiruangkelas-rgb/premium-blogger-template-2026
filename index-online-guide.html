📚 How to Use

Option 1: Standalone HTML (No Install)

Perfect for quick testing and simple blogs:

1. Download demo-standalone.html
2. Open in any web browser
3. Edit HTML directly with your content
4. Upload to any hosting service

Option 2: For Development

If you want build tools and advanced features:

git clone https://github.com/edukasiruangkelas-rgb/premium-blogger-template-2026.git
cd premium-blogger-template-2026
npm install
npm run dev

Option 3: For Blogger Platform

Direct installation on Blogger:

1. Go to Blogger Dashboard
2. Theme → Edit HTML
3. Copy paste the XML template
4. Save and customize

🔧 Customization (Easy!)

Change colors in the standalone HTML:

--color-primary: #7B3FF2; /* Change this to your color */
--color-primary-light: #9D6CFF;
--color-primary-dark: #6A2FB8;

Change fonts:

--font-primary: 'Raleway', sans-serif; /* Headings */
--font-body: 'Open Sans', sans-serif; /* Body text */
