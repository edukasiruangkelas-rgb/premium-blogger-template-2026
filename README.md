# Premium Blogger Template 2026

🎨 Template premium blog dengan dominan warna ungu dan putih, header scrolling, dan dukungan penuh untuk artikel dan tools.

## 🌟 Fitur Utama

- ✨ **Header Scrolling Responsif** - Smooth scrolling dengan animasi modern
- 🎨 **Desain Ungu & Putih Premium** - Warna scheme profesional (#7B3FF2 & #FFFFFF)
- 📱 **Fully Responsive** - Mobile-first design approach
- 🔧 **Tools & Articles Layout** - Dukungan penuh untuk grid tools dan blog posts
- ⚡ **Performance Optimized** - Fast loading dengan lazy loading & minified assets
- 🔍 **SEO Ready** - Schema.org markup, meta tags, open graph
- ♿ **WCAG 2.1 AA Compliant** - Accessibility first
- 📦 **Blogger Compatible** - XML template format

## 🚀 Quick Start

### Prerequisites
- Node.js 16+
- npm atau yarn
- Blogger account (untuk testing)

### Installation

```bash
# Clone repository
git clone https://github.com/edukasiruangkelas-rgb/premium-blogger-template-2026.git
cd premium-blogger-template-2026

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Generate Blogger template
npm run build:blogger
```
